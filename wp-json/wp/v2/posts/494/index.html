    \n
  1. 打开 Visual Studio Code，并在打开的项目文件夹中打开终端。<\/li>\n\n\n\n
  2. 在终端中输入以下命令创建虚拟环境，可以指定环境的名称和位置：<\/li>\n<\/ol>\n\n\n\n
       <\/span>python<\/span> <\/span>-m<\/span> <\/span>venv<\/span> <\/span><<\/span>env_nam<\/span>e<\/span>><\/span><\/span><\/code><\/pre>ShellScript<\/span><\/div>\n\n\n
    例如：<\/p>\n\n\n\n
       <\/span>python<\/span> <\/span>-m<\/span> <\/span>venv<\/span> <\/span>myenv<\/span><\/span><\/code><\/pre>ShellScript<\/span><\/div>\n\n\n
    这将在当前项目文件夹中创建一个名为 myenv<\/code> 的虚拟环境。<\/p>\n\n\n
      \n
    1. 虚拟环境创建完成后，需要激活虚拟环境。在终端中输入以下命令来激活虚拟环境：<\/li>\n<\/ol>\n\n\n
        \n
      • Windows: .\\myenv\\Scripts\\activate<\/code>\n<\/li>\n\n\n\n
      • macOS\/Linux: source myenv\/bin\/activate<\/code> 你会注意到终端提示符前面会出现虚拟环境的名称，表示虚拟环境已激活。<\/li>\n<\/ul>\n\n\n
          \n
        1. 现在你可以在激活的虚拟环境中安装包或执行 Python 脚本了。<\/li>\n<\/ol>\n\n\n
          注意：激活虚拟环境后，使用 python<\/code> 命令将使用虚拟环境中的 Python 解释器。如果要停止使用虚拟环境，可以在终端中使用 deactivate<\/code> 命令。<\/p>\n\n\n
