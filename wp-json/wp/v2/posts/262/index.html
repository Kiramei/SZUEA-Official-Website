在配置Wordpress设置的时候，有时会去更改地址端口号，就像下图：<\/p>\n\n\n


但是一旦更改后，如果没有正确配置，可能会导致站点无法访问，而且连登录wordpress都成困难。<\/strong><\/p>\n\n\n\n

\n\n\n

下面介绍还原回去的办法。<\/h3>\n\n\n\n

在Wordpress中，通过更改Wordpress的php文件是无法达到目的的，因为我们所更改的站点地址是存储于数据库<\/strong>中的，因此我们考虑去修改数据库中的站点地址。<\/p>\n<\/blockquote>\n\n\n

    \n
  1. 由于我使用的是Docker搭建的Wordpress，因此我首先要查看我的容器是什么名字，运行：docker ps来查询，找到类似下图的条目，我的数据库容器名是blog-db-1 。\n<\/li>\n\n\n\n
  2. 如果你不清楚你的数据库密码，就先不要进入这个容器，Wordpress里是有对你的账密作记录的，找到你的 WordPress 安装目录中的 wp-config.php 文件，对于我的docker来说，它在html文件夹下，并打开它。该文件包含连接数据库所需的登录信息，如数据库名称、用户名和密码。\n<\/li>\n\n\n\n
  3. 接着，执行docker exec -it blog-db-1 bash 进入docker命令行。然后输入mysql -u 你的用户名 -p ，回车并输入密码，进入mysql命令行。\n<\/li>\n\n\n\n
  4. 执行SHOW databases; 找到wordpress数据库，我的数据库就叫wordpress。\n<\/li>\n\n\n\n
  5. 执行USE wordpress ，然后执行下列语句，其中XXX 是你要更改回去的地址，注意协议名和端口号。
